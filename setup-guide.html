🚀 Portfolio Setup Guide

Get your professional freelance portfolio live in minutes

Open index.html in your web browser. That's it! Your site is ready to view.

Step 2: Customize Basic Info

Step 3: Setup Email

In script.js, find this line and replace with your email:

📧 Email Setup Guide

Option 1: FormSubmit.co (Recommended)

Why FormSubmit?
  • ✓ Completely free
  • ✓ No account needed
  • ✓ Takes 30 seconds to setup
  • ✓ GDPR compliant
Setup Steps:
  1. Open script.js
  2. Find: form.action = 'https://formsubmit.co/YOUR-EMAIL@example.com';
  3. Replace YOUR-EMAIL@example.com with your real email
  4. Save the file
  5. Test the contact form

Option 2: EmailJS (More Features)

Setup Steps:
  1. Go to emailjs.com
  2. Sign up for free
  3. Create an email service
  4. Get your Service ID, Template ID, and Public Key
  5. In script.js, uncomment the EmailJS section
  6. Add your credentials

🔧 Detailed Customization Guide

1. Update Your Profile

<h1>Hi, I'm Your Name</h1>
<p>Your Professional Title or Tagline</p>

2. Update Skills

Find the skills section and replace with your skills:

<span class="skill-tag">Your Skill</span>

3. Add Portfolio Projects

Edit or add new portfolio items:

<div class="portfolio-item">
    <div class="portfolio-image">
        <i class="fas fa-icon-name"></i>
    </div>
    <h3>Project Title</h3>
    <p>Project description</p>
    <span class="tech-stack">Tech Stack</span>
</div>

4. Change Colors

Edit styles.css and update these colors:

:root {
    --primary-color: #007bff;    /* Main color */
    --secondary-color: #0056b3;  /* Dark variant */
    --accent-color: #ffc107;     /* Highlight */
}

<p><a href="tel:+1234567890">+1 (234) 567-890</a></p>

6. Add Social Links

</a>

🌐 Deploy Your Portfolio

❓ Frequently Asked Questions

Q: Will my contact form work?

A: Yes! Make sure you've added your email to the FormSubmit line in script.js.

Q: Can I change the colors?

A: Absolutely! Edit the color codes in styles.css under the :root section.

Q: How do I add more projects?

A: Copy a portfolio-item div and change the title, description, and icon.

Q: Can I use my own domain?

A: Yes! Buy a domain and point it to your hosting provider (GitHub Pages, Netlify, etc).

Q: Is this mobile-friendly?

A: Yes! It's fully responsive and works on all devices.

Q: Can I add more sections?

A: Yes! Copy existing sections and customize them. The structure is flexible.

🔧 Troubleshooting

Contact form not sending emails?

  • ✓ Check that you updated the email in script.js
  • ✓ Check browser console for errors (F12 → Console)
  • ✓ Verify your email is formatted correctly
  • ✓ Try a different email service (EmailJS)

Website looks broken on mobile?

  • ✓ Clear browser cache (Ctrl+Shift+Delete)
  • ✓ Test in Chrome DevTools (F12 → Toggle device toolbar)
  • ✓ Check that all CSS files are in the same folder

Icons not showing?

  • ✓ Check internet connection (icons load from CDN)
  • ✓ Verify correct icon names at fontawesome.com
  • ✓ Refresh the page
